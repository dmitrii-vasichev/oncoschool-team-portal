# Changelog

Собран из change-фрагментов в `.changes/`.

## Business Changes

### 2026-02-21
- **ONCO-3X** (improvement, Рассылки Telegram, risk: low): В блоке участников для тегов единый список заменен на два сценария раскрытия: по отделам и по ролям; по умолчанию активен режим по отделам, группы в обоих режимах свернуты. | value: Выбор получателей тегов стал быстрее и понятнее для менеджеров: проще находить нужных людей в крупных командах и снижать ошибки при массовых рассылках.
- **ONCO-XXX** (improvement, Рассылки Telegram, risk: low): В форме создания рассылки добавлена вставка картинки из буфера обмена (Ctrl+V/Cmd+V); выбор файла через проводник сохранён, для обоих сценариев используется единая валидация типа и размера изображения. | value: Подготовка рассылок стала быстрее: можно вставлять свежие скриншоты напрямую без сохранения файла вручную.
- **ONCO-XXX** (fix, Встречи / Zoom, risk: medium): Исправили жизненный цикл Zoom для расписаний встреч: для «по мере необходимости» при назначении даты и при переводе в еженедельный режим Zoom-ссылка создается сразу после сохранения; при обратном переводе в «по мере необходимости» без даты дата и Zoom-ссылка очищаются, связанная Zoom-встреча удаляется; также устранена ошибка 500 при сохранении таких изменений. | value: Пользователи получают предсказуемое поведение встреч и Zoom без ручных обходов: ссылка появляется/исчезает сразу по логике сценария, а сохранение настроек выполняется стабильно.
- **ONCO-XXX** (improvement, Telegram бот / Задачи, risk: low): Создание задач из голосовых сообщений теперь доступно участникам; при этом выбор исполнителя в голосовом сценарии оставлен только для модераторов и дополнительно защищен серверной проверкой. | value: Задачи из голосовых фиксируются быстрее без привлечения модератора, при этом сохраняются корректные границы прав по назначению исполнителя.
